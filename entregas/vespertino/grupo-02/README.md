# Projeto Integrador: Banco de Dados e SQL
## Grupo 02 - Vespertino

Bem-vindo ao repositório do **Grupo 02**. Este projeto tem como objetivo desenvolver uma aplicação web de consulta acadêmica, utilizando **Streamlit**, **Python** e **MySQL** para manipular a base de dados `escola_db`.

### Tecnologias e Ferramentas
* **Linguagem**: Python
* **Interface**: Streamlit
* **Banco de Dados**: MySQL
* **Gerenciamento de Banco**: HeidiSQL (utilizado para modelagem, manutenção e visualização dos dados)
* **Backend/API**: FastAPI

### Estrutura do Repositório
Para facilitar a navegação e a avaliação, organizamos o projeto da seguinte forma:

* **/app**: Contém o código-fonte da aplicação web (interface e backend).
* **/sql**: Contém os scripts SQL para criação e população do banco `escola_db`.
* **/der**: Contém o Diagrama Entidade-Relacionamento (DER) do sistema.
* **/docs**: Documentação técnica e orientações do projeto.
* **/infra**: Arquivos de configuração de infraestrutura.
